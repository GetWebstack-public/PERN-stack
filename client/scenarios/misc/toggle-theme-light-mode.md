## Prerequisites
- User must have access to the web application at the URL: https://cafetime-demo.web.app/.
- User is logged into the application.
- The initial theme of the application is set to dark mode.

## Test Steps
- Open a web browser and navigate to the URL: https://cafetime-demo.web.app/.
- Confirm that the application is displayed in dark mode.
- Locate the theme toggle button, which is typically represented by a sun or moon icon.
- Click on the theme toggle button.
- Observe any visual changes in the interface as the theme switches.
- Verify that the application is now displayed in light mode.

## Expected Behaviour
- Upon clicking the theme toggle button, the application should transition from dark mode to light mode smoothly.
- The background color should change to a lighter shade, and text should adapt to ensure readability against the new background.
- Any graphical elements should also be updated to reflect the new light theme.
- The change in theme should persist during the session.
