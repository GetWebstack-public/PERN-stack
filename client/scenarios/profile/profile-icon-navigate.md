## Prerequisites
- The user must have a registered account on the platform.
- The user must be logged into the application.
- The application should be accessible at the URL: https://cafetime-demo.web.app/.
- The user interface should be fully loaded, displaying the homepage.

## Test Steps
- Step 1: Open a web browser and navigate to the URL: https://cafetime-demo.web.app/.
- Step 2: Ensure that you are logged in with your user credentials.
- Step 3: Locate the profile icon on the user interface (usually located in the top right corner of the page).
- Step 4: Click on the profile icon.
- Step 5: In the dropdown menu that appears, select the option labeled "Settings" or "Profile".
- Step 6: Observe the navigation as the settings page loads.

## Expected Behaviour
- Upon clicking the profile icon, a dropdown menu should appear.
- The item for "Settings" or "Profile" should be clearly visible and clickable.
- Upon selecting the "Settings" option, the user should be navigated to the settings page without any errors.
- The settings page should display options for updating profile information, confirming successful navigation.
